# NDA Project #1

Данный проект, выполненный в рамках коммерческой разработки под NDA.<br/>
Разработка велась по предоставленному ТЗ, дизайну и требованиям заказчика с использованием **PUG, SCSS** и **JavaScript** (**Gulp-сборка**, но в данном проекте не представлена).

В портфолио представлен переработанный фрагмент фронтенд проекта: код адаптирован из реальной рабочей версии с удалением логотипов и упоминаний бренда.
